# AI Dataset Analyzer using n8n

This project automates data analysis using n8n and AI APIs.

## Features
- Upload CSV dataset
- Clean data automatically
- Generate AI insights
- Send results via email

## Workflow
Webhook → Extract → Code → HTTP → Set → Gmail

## Tech Stack
- n8n
- JavaScript (Code Node)
- AI API (OpenRouter)

## Example
Tested using Titanic dataset.

## Output
The system analyzes dataset and sends insights via email.

## How to Use
1. Import the workflow JSON into n8n
2. Connect your API keys
3. Run the workflow
